# Installation

## Supported setups

Common setups include:
- Geyser Standalone
- Geyser on Velocity / BungeeCord
- Geyser as a plugin (Paper/Spigot/etc.)
- Geyser as a mod (Fabric/NeoForge/etc.)

## Install steps
[Follow this guide](https://geysermc.org/wiki/geyser/extensions#installing-extensions)

## Recommended

- Keep Geyser updated to the latest stable release.
- Use the extension release that matches your Geyser API version.
